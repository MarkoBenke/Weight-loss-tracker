# Weight Loss Tracker App
This is a clean architecture app example built with:

- Kotlin
- Coroutines
- MVVM
- Dependency injection with Hilt
- Retrofit
- Room
- Data Store
- Navigation Components
- LiveData
- Repository Pattern
- Coroutines Flow
- ViewBinding
- Glide
- Lottie Animations
- Extension Functions
